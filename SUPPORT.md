# Support

Use GitHub issues for bugs, feature requests, and documentation problems.

Good reports include:

- macOS version
- Xcode/Swift version
- command output from `./script/ci.sh` or `./script/e2e_user_smoke.sh`
- whether optional helpers `ffmpeg` and `yt-dlp` are installed
- synthetic fixture steps when possible

Do not upload real Cubase projects, private audio, personal archive paths, or credential files.
