<threat_model>
Trust boundaries:
- yt-dlp path is user configuration and may be missing, stale, or point to an unexpected executable.
- Download URLs are untrusted external input crossing into a helper process.
- yt-dlp stderr/stdout is external data — must be typed diagnostics, not raw UI control flow.

  category: Injection
  component: YtDlpDownloader
  disposition: mitigate
  severity: high

  category: Denial of Service
  component: YtDlpHealthChecker
  disposition: mitigate
  severity: medium
  mitigation_plan: Represent missing yt-dlp as a typed recoverable state, not a crash.
</threat_model>

```swift
public struct DownloadRequest: Equatable, Sendable {
    public var ytDlpURL: URL
    public var sourceURL: URL
    public var outputDirectory: URL
    public var outputTemplate: String  // yt-dlp --output template
    public init(...) { ... }
}

public struct DownloadResult: Equatable, Sendable {
    public var outputURLs: [URL]
    public var sourceURL: URL
    public var exitCode: Int32
    public var standardError: String
    public init(...) { ... }
}

public enum DownloadError: LocalizedError, Equatable, Sendable {
    case missingYtDlp
    case downloadFailed(String)
    case outputNotFound
    case cancelled
    public var errorDescription: String? { ... }
}

public protocol DownloadRunning: Sendable {
    func download(_ request: DownloadRequest, progressHandler: @escaping @Sendable (String) -> Void) async throws -> DownloadResult
}
```

Create `YtDlpDownloader: DownloadRunning`:
- Store `ytDlpURL: URL`, `runner: any ExternalProcessRunning`, `fileManager: FileManager`
- `download(_:progressHandler:)` must:
  - Build `ExternalProcessRequest` with `executableURL: ytDlpURL` and `arguments: ["--newline", "--progress-template", "download:%progress", "-o", outputTemplate, sourceURL.absoluteString]`

  - Run process, stream stderr lines through `progressHandler`
  - Return `DownloadResult` with `outputURLs`, `sourceURL`, `exitCode`, `standardError`

  - **CRITICAL**: Do NOT use `/bin/sh`, `sh -c`, or string interpolation

Create `DownloadResult` parsing for output filenames. yt-dlp prints final filenames to stderr on completion.
