<threat_model>
Trust boundaries:
- Local user launches the macOS app and chooses local files/folders in later plans.
- Phase 1 has no network, no downloader execution, and no FFmpeg/yt-dlp process execution.
- The shell must not expose fake disabled tools that imply unavailable behavior.

STRIDE register:
- threat_id: T-01-UI
  category: Spoofing
  component: AppShellView / ToolSidebarView
  disposition: mitigate
  severity: low
  mitigation_plan: Render sidebar rows only from ToolRegistry metadata and verify Dev Tool is the only Phase 1 tool.
- threat_id: T-01-CONFIG
  category: Tampering
  component: AppComposition
  disposition: mitigate
  severity: low
  mitigation_plan: Keep registration static at startup in AppComposition instead of global mutable self-registration.
- threat_id: T-01-LOG
  category: Information Disclosure
  component: DevToolFeature
  disposition: accept
  severity: low
  mitigation_plan: Dev Tool copy may show feature/context diagnostics, but it must not print full user paths or secrets in Phase 1.
</threat_model>
