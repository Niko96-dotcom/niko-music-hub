# Plan 29-01: Downloader UAT matrix, helper-path tests, and live smoke

## Goal

Close v1.4 with deterministic downloader UAT traceability, opt-in live `yt-dlp` verification beyond the 18-second happy path, stripped helper-path proof, and documented evidence.

## Tasks

### Task 1: Deterministic UAT coverage matrix

Add `DownloaderUATCoverageTests.swift` mapping UAT-01 areas to test types/files (progress, stall, helper health, structured output, media handoff). Add `DownloaderHelperToolResolverTests.swift` for PATH augmentation and ffmpeg parent resolution.

**Verify:** `swift test --filter DownloaderUATCoverageTests --filter DownloaderHelperToolResolverTests`

### Task 2: Opt-in live integration test

Add `DownloaderLiveIntegrationTests.swift` gated by `NIKO_MUSIC_HUB_LIVE_DOWNLOADER=1`. Uses real `YtDlpDownloader` against `BaW_jenozKc` audio-only; asserts progress markers and output file. XCTSkip when env unset or yt-dlp missing.

**Verify:** `NIKO_MUSIC_HUB_LIVE_DOWNLOADER=1 swift test --filter DownloaderLiveIntegrationTests` (when yt-dlp available)

### Task 3: Live smoke script + evidence doc

Add `script/downloader_live_smoke.sh` (env-gated) covering stripped PATH + `--ffmpeg-location`, simulate, download, progress grep, ffprobe duration > 18s, failure URL. Add `docs/downloader-v1.4-uat.md`. Wire `script/dev.sh live-downloader` helper.

**Verify:** `./script/ci.sh` and `./script/e2e_user_smoke.sh` green; script exits 0 when skipped without env var.

### Task 4: Phase verification

Write `29-VERIFICATION.md` with passed status after gates green.

**Verify:** `./script/ci.sh && ./script/e2e_user_smoke.sh`
