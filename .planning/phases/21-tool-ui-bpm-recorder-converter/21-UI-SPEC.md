# Phase 21 — UI Design Contract

**Scope:** BPM Tapper, Audio Recorder, WAV Converter  
**Canonical:** `docs/UI-REDESIGN-PLAN.md` §8–10

## BPM Tapper
- Centered column, 56pt hero BPM, segmented adjustment, 240pt+ tap pad, labeled Copy/Save/Reset, separator before history.

## Audio Recorder
- Centered hero timer 56pt, gradient meter 8px, large red Record/Stop, save toast 5s auto-dismiss.

## WAV Converter
- Dashed drop zone 180pt, pipe preset strip + Edit Preset, labeled Add/Convert/Stop, capsule badges on batch rows.

## Non-goals
Downloader, Settings, Archive surfaces (later phases).
